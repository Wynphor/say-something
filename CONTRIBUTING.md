# 投稿指南

## 📖 往壁报墙投稿

壁报墙是安静的角落，适合写得比较完整的文字。

### 步骤

1. Fork 本仓库
2. 在 `wall/` 对应目录下新建 `.md` 文件
3. 文件命名：`化名-日期.md`，例如 `阿鱼-20260603.md`
4. 提交 PR，系统会自动合并

### 文件内容模板

```markdown
# 标题（可选）

正文内容，自由发挥。

---
— 化名 / 网名
```

### 目录说明

| 目录 | 适合放什么 |
|------|-----------|
| `wall/whisper/` | 心事、碎碎念、深夜的胡思乱想 |
| `wall/story/` | 你的故事、你听来的故事、一段回忆 |
| `wall/unsaid/` | 一直想说但没说出口的话 |
| `wall/poetry/` | 诗、散文、文艺创作 |
| `wall/daily/` | 今天的天气、路上的猫、吃到好吃的东西 |
| `wall/gratitude/` | 想感谢的人、温暖的小事 |
| `wall/archive/` | 转载别人的文字（请注明出处） |

### 注意事项

- 自创和转载都可以，转载请注明来源
- 内容自由，没有字数要求
- 匿名或化名均可

---

## 💬 在回音壁发帖

回音壁是流动的评论区，适合想跟人聊的内容。

### 步骤

1. 点击 [新建 Issue](../../issues/new/choose)
2. 选择对应的模板（心事 / 故事 / 遗憾 / 日常 / 感恩）
3. 填写标题和内容，提交即可

### 关于评论

- 任何人可以在 Issue 下面留言
- 可以用 emoji reaction 表达共鸣
- 评论请遵守 [社区公约](CODE_OF_CONDUCT.md)

---

## 两者的区别

| | 壁报墙 | 回音壁 |
|--|--------|--------|
| 形式 | 仓库文件（PR） | Issue |
| 互动 | 可以评论 PR | 自由评论 |
| 节奏 | 安静、沉淀 | 即时、流动 |
| 适合 | 写好了的文字 | 想跟人聊 |
