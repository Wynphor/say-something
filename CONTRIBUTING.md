# 投稿指南 / How to Contribute

---

## 📖 往壁报墙投稿 / Submit to The Wall

壁报墙是安静的角落，适合写得比较完整的文字。

*The Wall is a quiet space — for words that feel finished.*

### 步骤 / Steps

```
1. Fork 本仓库
2. 在 wall/ 对应目录下新建 .md 文件
3. 文件命名：化名-日期.md（例：阿鱼-20260603.md）
4. 提交 PR，系统会自动合并
```

```
1. Fork this repo
2. Create a .md file in the right folder under wall/
3. Name it: alias-date.md (e.g. littlebird-20260603.md)
4. Open a PR — it'll be auto-merged
```

### 文件模板 / Template

```markdown
# 标题 / Title（可选 / optional）

正文。想写什么写什么。

Write whatever you want. No rules on length or style.

---
— 你的化名 / Your alias
```

### 目录说明 / Folder guide

| 目录 | 放什么 | English |
|:---:|:---|:---|
| `whisper/` | 心事、碎碎念 | Whispered thoughts, late-night ramblings |
| `story/` | 故事、回忆 | Stories, memories, things that happened |
| `unsaid/` | 没说出口的话 | Things you wanted to say but didn't |
| `poetry/` | 诗、散文、文艺创作 | Poetry, prose, anything literary |
| `daily/` | 天气、猫、好吃的 | Daily life — weather, cats, good food |
| `gratitude/` | 感谢、温暖的小事 | Gratitude, small kindnesses |
| `archive/` | 转载（注明出处） | Borrowed words (credit the source) |

---

## 💬 在回音壁发帖 / Post on Echo Wall

回音壁是流动的评论区，适合想跟人聊的内容。

*Echo Wall is a flowing comment section — for things you want to talk about.*

### 步骤 / Steps

```
1. 点击「新建 Issue」
2. 选择对应模板
3. 填写内容，提交
```

```
1. Click "New Issue"
2. Pick a template
3. Write and submit
```

评论区自由留言，用 ❤️ 表达共鸣。

*Anyone can comment. Leave a ❤️ if it resonates.*

---

## 两者的区别 / The difference

| | 📖 壁报墙 / The Wall | 💬 回音壁 / Echo Wall |
|:---:|:---|:---|
| 形式 / Form | 仓库文件 / Repo files | Issue |
| 互动 / Interaction | 可以评论 PR / Comment on PR | 自由评论 / Open comments |
| 节奏 / Pace | 安静、沉淀 / Quiet, settled | 即时、流动 / Live, flowing |
| 适合 / Best for | 写好了的文字 / Polished writing | 想跟人聊 / Starting a conversation |

---

## 🤝 注意事项 / Notes

- 自创和转载都可以，转载请注明来源
- 内容自由，没有字数要求
- 匿名或化名均可

---

- Original or borrowed — both welcome. Just credit the source.
- No word count. No format requirements. Write what feels right.
- Anonymous or alias. Up to you.
