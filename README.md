## 这是什么

一个基于 GitHub 的善意角落。

不是论坛，不是社交平台，而是一面墙。每个人都走过来，写下一句话、一段故事、一个念想，然后离开。后来的人路过，读到了，觉得被触碰了一下。

这里有少女心事，有美好的遗憾，有但行好事莫问前程。

玩梗是为了博人一乐，文艺是为了寻找共鸣，自创转载，皆为华章。

## 两个空间

### 📖 壁报墙

> 像贴在墙上的纸条，路过的人驻足阅读。

安静的、沉淀的、经过整理的 —— 像一本翻开的书。

**投稿方式：** Fork 本仓库 → 在 `wall/` 对应目录新建 `.md` 文件 → 提交 PR

文件提交后会**自动合并上墙**，不设审批门槛。

适合写得比较完整、希望被更多人看到的文字。

| 目录 | 说明 |
|------|------|
| `wall/whisper/` | 心事 & 碎碎念 |
| `wall/story/` | 故事 & 回忆 |
| `wall/unsaid/` | 没能说出口的话 |
| `wall/poetry/` | 诗 & 文艺创作 |
| `wall/daily/` | 日常碎片 |
| `wall/gratitude/` | 感谢 & 温暖 |
| `wall/archive/` | 转载的触动文字 |

### 💬 回音壁

> 像深夜的评论区，你说了，有人听见。

即时的、流动的、有回应的 —— 像一封信寄出去了，真的有人回。

**参与方式：** 直接开一个 Issue，选择对应模板，写下你想说的。

别人可以在底下留下评论，用 ❤️ 表达无声的共鸣。

## 参与方式

想说点什么？→ [开一个 Issue](../../issues/new/choose)

想贴上墙？→ [提一个 PR](../../pulls)（会自动合并）

想安静地读？→ 随意逛，不需要做任何事。

---

## 我们的约定

这里只有善意。

不争论对错，不阴阳怪气，不说教评判。

你可以质朴，可以文艺，可以幽默，可以沉默。

但请尊重每一个人写下的文字，就像尊重你自己的心事。

详见 [社区公约](CODE_OF_CONDUCT.md)。

## 投稿格式参考

壁报墙文件命名：`化名-日期.md`，例如：

```
一个路过的猫-20260603.md
```

文件内容自由，可以参考 [投稿指南](CONTRIBUTING.md)。
