# How to Contribute

---

## 📖 Submit to The Wall

The Wall is a quiet space — for words that feel finished.

### Steps

1. Fork this repo
2. Create a `.md` file in the right folder under `wall/`
3. Name it: `alias-date.md` (e.g. `littlebird-20260603.md`)
4. Open a PR — it'll be auto-merged

### Template

```markdown
# Title (optional)

Write whatever you want. No rules on length or style.

---
— Your alias
```

### Folder guide

| Folder | What goes here |
|:---:|:---|
| `whisper/` | Whispered thoughts, late-night ramblings |
| `story/` | Stories, memories, things that happened |
| `unsaid/` | Things you wanted to say but didn't |
| `poetry/` | Poetry, prose, anything literary |
| `daily/` | Daily life — weather, cats, good food |
| `gratitude/` | Gratitude, small kindnesses |
| `archive/` | Borrowed words that moved you (credit the source) |

---

## 💬 Post on Echo Wall

Echo Wall is a flowing comment section — for things you want to talk about.

### Steps

1. Click "New Issue"
2. Pick a template
3. Write and submit

Anyone can comment. Leave a ❤️ if it resonates.

---

## The difference

| | 📖 The Wall | 💬 Echo Wall |
|:---:|:---|:---|
| Form | Repo files (PR) | Issue |
| Interaction | Comment on PR | Open comments |
| Pace | Quiet, settled | Live, flowing |
| Best for | Polished writing | Starting a conversation |

---

## Notes

- Original or borrowed — both welcome. Just credit the source.
- No word count. No format requirements. Write what feels right.
- Anonymous or alias. Up to you.
